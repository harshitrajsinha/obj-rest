## Purpose of creating the project

* To learn REST API implementation by creating a backend server to serve data from external API - https://restful-api.dev/.
* Implement API security.
* Extend the endpoints beyond CRUD operations - search
* API performance measurement and implementation.

### Implementation to serve data from external source

* `/store` defines methods that will be invoked to retrieve data
* `interface.go` contains those method signature that will be implemented for API call and testing
